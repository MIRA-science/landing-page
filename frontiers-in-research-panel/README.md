Companion **post-show notes** for the Catalyze *Frontiers in Research: Open Science* panel (June 18, 2026): a question-and-answer recap of the conversation, not an official transcript.

- **Live page:** https://mira-science.github.io/landing-page/frontiers-in-research-panel/
- **Recording:** https://www.youtube.com/watch?v=b5SQuNGSdlY

## Contents

- `index.html` — the page itself: a single self-contained file with the embedded recording, an interactive Q&A accordion, verified quotes, and canonical links.
- `how-this-was-made/` — the build record:
  - `conversation_log.md` — the prompts and steps that produced the page (the provenance trail).
  - `README.md` — orientation for the build log.

## Notes

- **Unofficial.** Compiled by the Discourse Graphs team, not the panel organizers. Block quotations are drawn from the recording and lightly edited for readability (speech-to-text fixes only); audience questioners are shown by first name + last initial.
- **Raw source materials** (full transcript, Zoom chat, attendee list/photo) are kept local and are **not** included here, since they contain attendee names and images.
- A Notion mirror is used for panelist comments and edits; approved changes are folded back into `index.html`.
